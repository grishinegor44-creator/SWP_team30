## Description of MVP v0

Mvp v0 is frontend only website, deployed to special ngrok like tool to be accessible from global internet. For now mvp v0 doesn't contain any backend logic.

## Relationship to prototype

Current mvp v0 is prototype with simplified UI and simple behavoiur (event registration form gives success message)

## Relationship to user stories

MVP-v0 satisfies US-10 about welcome page. Other US requires some backend powered functionality

## Current limitations, placeholders, and mocks

Current mvp contains mocks inside all text and image fields, all existing game cards and statistics are placeholders. Current mvp v0 completely disconnected from backend, so its limited to only simple behaviour
